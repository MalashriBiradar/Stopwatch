A simple and interactive Stopwatch built using HTML, CSS, and JavaScript.
This project demonstrates time calculation, DOM manipulation, and event handling in JavaScript.
<br>
Feature:

1. Start the Stopwatch
2. Pause/Resume timing
3. Reset the Stopwatch
4. Lap recording with multiple laps
5. Display tme in HH:MM:SS format
6. Accessible UI using aria-label and regions

Technologies Used

HTML5 - Structure of the app
<br>
CSS3 - styling and layout
<br>
JavaScript(ES6) - Stopwatch logic and time calculation
